# 井字棋静态网页

一个纯静态、可直接双击打开的井字棋游戏（Tic-Tac-Toe）。支持：
- 玩家 vs 玩家 (本地双人)
- 玩家 vs AI（简单 / 中级 / 困难）
- 统计胜局 / 平局次数
- 先手切换 (X 或 O)
- 深色模式切换

## 运行方式
无需任何构建或服务器。
1. 打开文件夹 `jingziqi`
2. 双击 `index.html` 用浏览器打开（建议 Chrome / Edge / Firefox）。

## 文件结构
```
jingziqi/
  index.html   # 页面骨架
  style.css    # 样式与主题
  script.js    # 游戏逻辑与 AI
  README.md    # 说明文档
```

## AI 难度说明
- 简单：随机选择一个可落子位置
- 中级：有限深度的 Minimax + 估值函数（速度更快）
- 困难：完整 Minimax 搜索（井字棋状态空间很小，可瞬间计算）

## 可扩展方向
